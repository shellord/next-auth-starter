## Watashi-app

A WebApp to customise a profile with interests and hobbies and share with public.

## TechStack

- NextJS
- TypeScript
- React
- TailWindCSS
- Prisma
- PlanetScale

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
